unzip and extrac the vappc_2012 csv files to the data directory

place the costoftobacco html file into the template directory

feel free to rename the app_aalvarez.py file to app.py 
or just run it as is.


NOTE: there is an odd resizing issue so in order for the browser to render
the visualizations properly, AFTER selecting a year, just maximize(fullscreen) and resize your browser.
They do work though, tested on Chrome and Internet explorer.

Image file is just for page preview.
